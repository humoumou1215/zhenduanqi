# 诊断器 (Zhenduanqi)

## 项目简介
Arthas 远程诊断工具 - 支持场景化诊断向导、RBAC 权限管理、操作审计的 Web 管理平台

## 功能特性
- 场景化诊断向导
- RBAC 权限管理
- 操作审计日志
- Arthas 命令执行
- 诊断结果可视化

## 技术栈
- 后端: Spring Boot
- 前端: Vue 3 + Element Plus
- 数据库: H2/MySQL

## 快速开始
### 环境要求
- Java 17+
- Node.js 16+
- Maven 3.8+

### 启动项目
1. 后端启动
```bash
./mvn-java17.sh spring-boot:run
```

2. 前端启动
```bash
cd frontend
npm install
npm run dev
```
